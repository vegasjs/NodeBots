NodeBots Vegas is all about experimenting at the intersection of hardware and software:

  • We don't normally have speakers - just bring your hardware and your laptop!
  • We talk about everything from blinking LEDs with Arduinos and Raspberry Pis to drones, robots, wearables and more!
  • Interested, but no idea how to get started? Check out the resources tab first, then come and we'll show you the ropes.
  • JavaScript is not required - all languages are welcome. Want to hack in Ruby, Python, Lua, C or Java? No problem!

We meet on the first Tuesday of every month.

Want to get started with JavaScript and Hardware? Here are some of our favorite resources:

  • NodeBots.io - Find other local NodeBots meetups

Tutorials

  • node-ardx - tutorials for using using an Arduino with node and various components
  • nodebots workshop - a tutorial in the nodeschool 'workshopper' format that requires a bit of terminal knowledge to get started.
  • sumobot tutorial - slightly dated video tutorial on assembling and programming a sumobot
  • Adafruit Learning System - a lot of tutorials for products available at Adafruit

Software

  • Johnny-Five - a high level JavaScript library for working with hardware

Microcontrollers

  • Arduino - this is the most ubiquitous and popular microcontroller. It is supported by Johnny-Five and a lot of other software libraries for hardware.
  • Raspberry Pi - not really a microcontroller, but a tiny computer that can be used to talk to hardware via special GPIO pins. You will need the raspi-io plugin for Johnny-Five.
  • Particle Photon - an inexpensive ($20) wifi capable microcontroller that you can control from Johnny-Five wirelessly using the particle-io plugin and the voodoospark firmware on the device.

Robots and Drones

  • NodeCopters - control drones from JavaScript!
